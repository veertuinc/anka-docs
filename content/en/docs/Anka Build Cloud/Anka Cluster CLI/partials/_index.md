```shell
> sudo ankacluster  --help
ankacluster starts or stops anka agent and connects or disconnects to an Anka cluster

Usage:
  ankacluster [flags]
  ankacluster [command]

Available Commands:
  disjoin     Disconnects this machine from the Anka Cloud cluster
  help        Help about any command
  join        Joins the current machine to Anka Cloud cluster
  run         Runs the Anka agent and prints it's output to stdout
  status      Returns the status of Anka Agent
  test        Tests connection from this machine the cluster

Flags:
  -h, --help      help for ankacluster
  -v, --version   show version and exit

Use "ankacluster [command] --help" for more information about a command.
```
