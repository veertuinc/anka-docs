```shell
> anka license activate --help
usage: activate [options] key

   Activate license key

arguments:
  key                      Activation key or license file

options:
  -f,--force               Silently override the current license
  --cid <val>              Submit the CustomerID
  --hostid <val>           Use alternate host id
  --cores <val>            Activate certain CPU cores
  -o,--output <val>        Save the license file at alternate path
```
